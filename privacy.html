Privacy Policy

Last updated: June 18, 2025

1. Introduction

TPKX TAG ("the App") is an internal-use tool that accesses TikTok's Research API. This Privacy Policy explains how data accessed through the API is handled.

2. Data We Access

The App accesses only publicly available TikTok data through the official Research API, including:

  • Public video metadata (views, likes, comments, shares, post date)
  • Public hashtag information
  • Public creator usernames associated with videos

3. Data We Do NOT Collect

  • Personal information of TikTok users
  • Private messages or non-public content
  • Device information, location, or behavioral data of any user
  • Any data from users who interact with this App (as it has no end-users)

4. How Data Is Used

All data retrieved via the API is used solely for:

  • Internal hashtag trend analysis
  • Competitor research reporting

Data is stored locally on the developer's device and is never sold, shared, or published.

5. Data Retention

Retrieved data is stored temporarily for analysis purposes and deleted when no longer needed. No long-term personal data database is maintained.

6. Third-Party Services

This App uses TikTok's Research API. Use of this API is governed by TikTok's Privacy Policy.
